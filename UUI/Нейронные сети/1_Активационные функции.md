| –ù–∞–∑–≤–∞–Ω–∏–µ  | –ö–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç                                    | –ü—Ä–∏–º–µ—Ä –≤—ã–∑–æ–≤–∞          | –ü—Ä–æ—Å—Ç–æ–π —Å–º—ã—Å–ª                                   |

| `sigmoid` | –æ—Ç 0 –¥–æ 1, –∫–∞–∫ S-–∫—Ä–∏–≤–∞—è                         | `activation='sigmoid'` | –í—ã—Ö–æ–¥ ‚Äî –≤–µ—Ä–æ—è—Ç–Ω–æ—Å—Ç—å                             |

## üé® –í–∏–∑—É–∞–ª–∏–∑–∞—Ü–∏—è —Ñ—É–Ω–∫—Ü–∏–π (Colab-—è—á–µ–π–∫–∞)

```python
import numpy as np
import matplotlib.pyplot as plt
import tensorflow as tf

x = np.linspace(-5, 5, 100)

# –ê–∫—Ç–∏–≤–∞—Ü–∏–æ–Ω–Ω—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏
relu = tf.nn.relu(x)
sigmoid = tf.nn.sigmoid(x)
tanh = tf.nn.tanh(x)

plt.figure(figsize=(10, 6))
plt.plot(x, relu, label='ReLU', linewidth=2)
plt.plot(x, sigmoid, label='Sigmoid', linewidth=2)
plt.plot(x, tanh, label='Tanh', linewidth=2)
plt.axhline(0, color='gray', lw=0.5)
plt.axvline(0, color='gray', lw=0.5)
plt.title("–ì—Ä–∞—Ñ–∏–∫–∏ –ø–æ–ø—É–ª—è—Ä–Ω—ã—Ö –∞–∫—Ç–∏–≤–∞—Ü–∏–æ–Ω–Ω—ã—Ö —Ñ—É–Ω–∫—Ü–∏–π")
plt.legend()
plt.grid(True)
plt.show()
```
