- –û—Ç—Ñ–∏–ª—å—Ç—Ä—É–π—Ç–µ –∑–∞–∫–∞–∑—ã, –æ–ø–ª–∞—á–µ–Ω–Ω—ã–µ —Å –ø–æ–º–æ—â—å—é "Apple Pay" –∏–ª–∏ "PayPal".

```py
print(df[df['—Å–ø–æ—Å–æ–±_–æ–ø–ª–∞—Ç—ã'].isin(['Apple Pay', 'PayPal'])])
```

### 3. **–ì—Ä—É–ø–ø–∏—Ä–æ–≤–∫–∞ –∏ –∞–≥—Ä–µ–≥–∞—Ü–∏—è**
- –ü–æ—Å—á–∏—Ç–∞–π—Ç–µ –æ–±—â—É—é —Å—É–º–º—É –≤—ã—Ä—É—á–∫–∏ –ø–æ –∫–∞–∂–¥–æ–π –∫–∞—Ç–µ–≥–æ—Ä–∏–∏ —Ç–æ–≤–∞—Ä–∞.
- –û–ø—Ä–µ–¥–µ–ª–∏—Ç–µ —Å—Ä–µ–¥–Ω—é—é —Ü–µ–Ω—É –∑–∞–∫–∞–∑–∞ –¥–ª—è –∫–∞–∂–¥–æ–≥–æ –∫–ª–∏–µ–Ω—Ç–∞.
- –ù–∞–π–¥–∏—Ç–µ –∫–ª–∏–µ–Ω—Ç–∞ —Å –Ω–∞–∏–±–æ–ª—å—à–∏–º –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ–º –∑–∞–∫–∞–∑–æ–≤.



### 4. **–°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞**
- –û—Ç—Å–æ—Ä—Ç–∏—Ä—É–π—Ç–µ –∑–∞–∫–∞–∑—ã –ø–æ –¥–∞—Ç–µ (–æ—Ç –Ω–æ–≤—ã—Ö –∫ —Å—Ç–∞—Ä—ã–º).
- –í—ã–≤–µ–¥–∏—Ç–µ —Ç–æ–ø-3 —Å–∞–º—ã—Ö –¥–æ—Ä–æ–≥–∏—Ö –∑–∞–∫–∞–∑–æ–≤.
- –û—Ç—Å–æ—Ä—Ç–∏—Ä—É–π—Ç–µ –∫–ª–∏–µ–Ω—Ç–æ–≤ –ø–æ –æ–±—â–µ–π —Å—É–º–º–µ –ø–æ—Ç—Ä–∞—á–µ–Ω–Ω—ã—Ö –¥–µ–Ω–µ–≥ (–ø–æ —É–±—ã–≤–∞–Ω–∏—é).

```py
# –°–æ—Ä—Ç–∏—Ä–æ–≤–∫–∞ –ø–æ –¥–∞—Ç–µ (–Ω–æ–≤—ã–µ ‚Üí —Å—Ç–∞—Ä—ã–µ)

# –ö–ª–∏–µ–Ω—Ç—ã –ø–æ —É–±—ã–≤–∞–Ω–∏—é —Å—É–º–º—ã –ø–æ–∫—É–ø–æ–∫
```

### 5. **–î–æ–±–∞–≤–ª–µ–Ω–∏–µ –Ω–æ–≤—ã—Ö —Å—Ç–æ–ª–±—Ü–æ–≤**
- –î–æ–±–∞–≤—å—Ç–µ —Å—Ç–æ–ª–±–µ—Ü "–û–±—â–∞—è —Å—Ç–æ–∏–º–æ—Å—Ç—å" (—Ü–µ–Ω–∞ * –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ).
- –°–æ–∑–¥–∞–π—Ç–µ —Å—Ç–æ–ª–±–µ—Ü "–ì–æ–¥ –∑–∞–∫–∞–∑–∞" –Ω–∞ –æ—Å–Ω–æ–≤–µ –¥–∞—Ç—ã.
- –î–æ–±–∞–≤—å—Ç–µ —Å—Ç–æ–ª–±–µ—Ü "–°–∫–∏–¥–∫–∞" (–Ω–∞–ø—Ä–∏–º–µ—Ä, 10% –¥–ª—è –∑–∞–∫–∞–∑–æ–≤ –¥–æ—Ä–æ–∂–µ 50).

```py
# –û–±—â–∞—è —Å—Ç–æ–∏–º–æ—Å—Ç—å (—Ü–µ–Ω–∞ * –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ)
df['–æ–±—â–∞—è_—Å—Ç–æ–∏–º–æ—Å—Ç—å'] = df['—Ü–µ–Ω–∞'] * df['–∫–æ–ª–∏—á–µ—Å—Ç–≤–æ']

# –ì–æ–¥ –∑–∞–∫–∞–∑–∞

# –°–∫–∏–¥–∫–∞ 10% –¥–ª—è –∑–∞–∫–∞–∑–æ–≤ –¥–æ—Ä–æ–∂–µ 50
```
